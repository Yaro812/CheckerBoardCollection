## Synopsis

UICollectionView vertical flow layout that lays cells in a checker board style

## Installation

Using Cocoapods: 
`pod 'CheckerBoardCollection'`

## Usage

To use automatic cell size calculation set property _columns_ to some value > 0
`layout.columns = 3`
